Introducción a CSS

    CSS (Cascading Style Sheets)
  • CSS es un lenguaje para descubrir la presentación de las páginas web, incluye color,diseño,fuentes,etc.
  • Permite adaptar la presentación a diferentes tipos de dispositivos tales como pantallas grandes , pantallas pequeñas e impresoras.
  • CSS es independiente de HTML, puede ser usado con cualquier lenguaje de marcado basado en XML.

    • La separación de HTML y CSS hace más fácil mantener sitios, compartir hojas de estilos entre páginas y ajustar las a diferentes entornos.
