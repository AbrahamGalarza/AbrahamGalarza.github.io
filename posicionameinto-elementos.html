Posicionamiento de elementos

Para poder posicionar los elementos existen 5 formas que son los siguientes

  1. Flujo Normal(position : static)
  2. Posicionamiento relativo (position : relative)
  3. Posicionamiento obsoluto
  4. Posicionamiento fijo (position : fixed)
  5. Elementos Flotantes

Flujo Normal (position : stactic)

Elementos en bloque uno encima de otro.

Elementos en linea de izquierda de derecha

Posicionamiento relativo (position : relative)

Este es relativo al elemento que lo contiene.

  • top
  • left
  • right
  • bottom

Posicionamiento Absoluto

  • Absoluto respecto al documento.
  • Se desplaza conforme al documento, se hace scroll.
