Propuesta de Proyecto

Página de Venta de Regalos

La página web tratara sobre una página que ofrece servicio de regalos
En la página se mostrarán todas las opciones posibles de regalos con su costo
que el usuario puede comprar. Los regalos son enviados a domicilio
para poder hacer una compra necesita estar registrado en la página,
en donde se le pedirá datos personales como nombre, edad, email, etc.
Cuando el usuario decida que regalo comprara y confirme que comprara,
se le pedirán la dirección a donde será enviado el regalo y sele
notificara el costo del producto.
