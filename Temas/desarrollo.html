Tipos de desarrollo

FrontEnd
El frontend son todas aquellas tecnologías que corren del lado del cliente, es decir, todas aquellas tecnologías que corren del lado del navegador web, generalizandose mas que nada en tres lenguajes, HTML , CSS Y JavaScript
BackEnd
El programador backend es aquel que se encuentra del lado del servidor, es decir, esta persona se encarga de lenguajes como PHP, Python, .Net, Java, etc, es aquel que se encarga de interactuar con bases de datos, verificar manejos de sesiones de usuarios, montar la página en un servidor, y desde este “servir” todas las vistas que el FrontEnd crea, es decir, uno como backend se encarga mas que nada de la manipulación de los datos
FullStack
Estos programadores se pueden encargar de los los tipos anteriores, el FrontEnd y el BackEnd. No solomanete de uno en especifico.
