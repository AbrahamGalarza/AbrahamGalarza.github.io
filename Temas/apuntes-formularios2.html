Formularios

Controles más Comunes

Texto

Linea de texto (text)
Este control permite capturar un texto, como por ejemplo un nombre.
Contraseña (password)
Este control permite capturar una contraseña, ya que al momento de introducirla, la contraseña queda encriptada en la pantalla.
Area de texto (textarea)
define un control de entrada de texto de varias líneas.Un área de texto puede contener un número ilimitado de caracteres.

Elecciones

Botones de radio (radio)
Representan de forma predeterminada como pequeños iconos circulares que se seleccionan cuando se activan. Los botones de radio le permiten seleccionar un solo valor de un número de opciones de conjunto para su presentación en un formulario.
Cajas de verificación (checkbox)
Permite insertar varias opciones como especies de casillas que se marcan o seleccionan. Tiene un atributo value que es usado para definir el valor enviado por el checkbox.
Cuadros de lista (Combobox)
Permite crear una lista de opciones desplegable. para definir este se utiliza el elemento select y dentro de esta etiqueta se agregan las opciones con la etiqueta "option.

Tiempo

Sirve para establecer referencia de tiempo como:

  • Fecha
  • Mes
  • Semana
  • Hora

Archivos (file)

Permiten al usuario escoger uno o más archivos como imagenes,documentos,etc., cargarlos en un servidor.

Botones (submit)

Un botón nos permite realizar una acción con el formulario que lo contenga. Existen dos tipos básicos de botones: botón enviar el formulario (envío o submit) y botón poner todos los campos a sus valores iniciales (restablecimiento o reset). Normalmente un formulario contendrá, como mínimo, un botón (el botón enviar).

Para incluir estos se debe agregar un elemento input y con el atributo type que tipo de control se desea. Para los cuadros de lista se utiliza la etiqueta select.
