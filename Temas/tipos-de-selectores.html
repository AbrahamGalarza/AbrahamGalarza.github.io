Selector Universal

Aplica un estilo a todos los elementos del documento.
ejemplo:
*{ font-type : Arial; }

Selector de tipo etiqueta

Aplica un estilo a todos los elementos de la etiqueta.
ejemplo:
h1{ font-size : 30px ; }

Selector de Clase

Aplica un estilo a los elementos que tengan el atributo class especificado por el valor.
ejemplo:
.nota { color : red ; }

Selector de Identificador

Aplica el estilo a los elementos que tienen el atributo id con con un valor dado.
ejemplo :
introduccion { background : cyan ; }

Selector hijo

Aplica el estilos a los elementos que son hijos inmediatos> de otro.
ejemplo:
li > a { font-weight : bold ;}

Selector Descendiente

Aplica el estilo a los elementos que son hijos descendientes (no necesariamente independientes).
ejemplo:
p a { texto-align : center ;}
