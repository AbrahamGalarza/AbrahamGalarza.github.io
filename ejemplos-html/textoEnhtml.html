Texto en HTML

Para poder agregar texto en lenguaje HTML tenemos los siguientes elementos

Ecabezados
Sirven para dividir el texto en secciones. Se pueden definir seis niveles de titulares, siendo "h1" el de mayor tamaño y el "h6" el de menor tamaño.
Párrafos
El elemento p (párrafo) es el apropiado para distribuir el texto en párrafos.
Saltos de linea
Este elemento realiza un salto de línea, podemos poner tantos como necesitemos y realizará un salto de línea por cada una de ellos
Negritas
Indica que el texto debe ser representado con una variable bold, o negrita, de la tipografía que se esté usando.
Cursiva
Muestra el texto marcado con un estilo en cursiva o italica.
Énfasis
Es el apropiado para marcar con énfasis las partes importantes de un texto.
Resaltar
Es el apropiado para marcar con especial énfasis las partes más importantes de un texto

Listas en HTML

Lista Ordenada
Sirven para colocar una lista numeración,su etiqueta es "ol"
Lista No Ordenada
Sirven para colocar una lista con viñetas, su etiqueta es "ul"
Lista de Definición
Sirven para colocar una lista de palabras con definiciones, su etiqueta es es "dl"
