Estructura de una tabla

  • Para definir una tabla se utiliza la etiqueta table
  • Para definir un renglon se utiliza la etiqueta tr
  • Para definir una celada en un renglon se utiliza la etiqueta td

Primero se debe definir la tabla con la etiqueta table dentro de la etiqueta table se define el renglon con la etiqueta tr y las celdas se define con la etiqueta td dentro de la etiqueta tr. Las celdas se van asignando de izquierda a derecha.
