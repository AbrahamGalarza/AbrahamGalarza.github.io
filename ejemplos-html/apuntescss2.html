CSS

CSS se escribe por medio de reglas, ejemplo de una regla de CSS
h1 { color : blue; }

Para agregar un comentario se agrega "/*" al inicio del comenrario y "*/" al terminar el comentario, por ejemplo.
/*Este es un comenetario*/

Tipos de Estilos

  1. Por default (aplicado por el navegador).
  2. Definidos por el usuario ("user styles")
  3. Hojas de estilos externos (Archivos independientes de HTML)
  4. Estilo Interno (etiqueta style de HTML)
  5. Estilo en linea (atributo style de HTML)
