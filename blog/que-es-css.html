¿Qué es CSS?

CSS es el acrónimo de Cascading Style Sheets, es decir, hojas de estilo en cascada. CSS es un lenguaje de estilo que define la presentación de los documentos HTML, como las fuentes, colores, márgenes, línea, altura, anchura, imágenes de fondo, posicionamiento, entre otros.
Es posible usar HTML, para añadir formato a los sitios web, pero CSS ofrece muchas más opciones y es más preciso y sofisticado. La diferencia entre HTML y CSS es que HTML se utiliza para estructurar el contenido y CSS se utiliza para formatear el contenido previamente estructurado.

Esta tecnología fue desarrollada por el World Wide Web Consortium (W3C) con el fin de separar la estructura de la presentación. A pesar de que la recomendación oficial del grupo de trabajo de la W3C ya había alcanzado la estabilidad requerida para que fuera soportada por los principales navegadores comerciales, como Netscape e Internet Explorer, tan tempranamente como en el año 1998, la situación de entonces, comúnmente conocida como la “guerra de los navegadores”, hacía que los intereses comerciales de las dos compañías en lucha por el mercado de usuarios de Internet se interpusieran en el camino de las CSS.
