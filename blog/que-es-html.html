¿Qué es HTML?

HTML es el lenguaje con que el que se crea el contenido de las páginas web, este se trata de un conjunto de etiquetas que sirven para definir el texto y los demás elementos que van a componer una página web, como imágenes, videos, listas, enlaces hacia otras páginas, etc.
HTML se creó con el principal objetivo divulgativo de información con texto y algunas imágenes. HTML se creó sin dar respuesta a todos los posibles usos que se le iban a dar y a todas las personas que lo utilizarían en un futuro. El significado de sus siglas en ingles es "HyperText Markup Language", es decir, Lenguaje de Marcas Hipertexto.

HTML es un lenguaje de marcación de elementos para la creación de documentos hipertexto, es muy fácil de aprender, por lo que permite que cualquier persona, aunque no tenga conocimientos en programación pueda desarrollar una web. Este lenguaje se escribe en un documento de texto, por lo que se necesita de un editor de textos para poder escribir el lenguaje HTML para nuestra página. A este archivo debe contener la extensión “.html” .

El lenguaje HTML es un estándar reconocido en todo el mundo y cuyas normas define un organismo sin ánimo de lucro llamado World Wide Web Consortium, más conocido como W3C. Como se trata de un estándar reconocido por todas las empresas relacionadas con el mundo de Internet, una misma página HTML se visualiza de forma muy similar en cualquier navegador de cualquier sistema operativo.
